# Software pro kategorizovanou evidenci osobních výdajů s využitím OCR a AI

## Přidělené téma

Toto téma je řešeno jako diplomová práce na Katedře informatiky. Téma se zabývá vývojem mobilní aplikace pro kategorizovanou evidenci osobních výdajů s využitím technologií Optical Character Recognition (OCR) a umělé inteligence (AI).

### Cíl práce

Cílem této diplomové práce je navázat na předchozí výsledky a rozšířit mobilní aplikaci pro kategorizovanou evidenci osobních výdajů. Důraz je kladen na vylepšení přesnosti OCR, rozšíření AI modelu pro kategorizaci a případně další funkcionality (např. prediktivní analýzy, více měn, sdílené rozpočty apod.).

### Osnova práce

#### Teoretická část

1. Analýza pokročilých přístupů v OCR a AI pro extrakci informací z účtenek.
2. Srovnání stávajících řešení a identifikace jejich limitů.
3. Popis použitých technologií a nástrojů.

#### Praktická část

1. Návrh vylepšené architektury aplikace a databáze s ohledem na škálování.
2. Implementace rozšíření aplikace a backendu včetně OCR a AI modulů.
3. Vytvoření a vyhodnocení experimentů pro měření přesnosti a výkonnosti.
4. Integrační a uživatelské testování.

### Použité technologie

- React Native
- Java Spring framework
- PostgreSQL
- OCR
- AI technologie
